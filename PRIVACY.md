# Privacy Policy

MATHAMOTA is a private chat application intended for direct use by the owner and the owner's invited partner.

## Data collected

- account display names
- chat messages
- file and media uploads
- chat preferences such as wallpaper and gallery visibility
- admin audit logs

## How data is used

- to deliver real-time chat
- to store and display shared content
- to provide owner-only admin visibility
- to maintain service security and debugging logs

## Important notice

This application is designed so that the owner/admin can view platform data. Do not describe it as end-to-end encrypted private messaging unless you change the architecture to remove server-side message visibility.

## Retention

Data remains stored until removed by the operator.
